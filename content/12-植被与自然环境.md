# 第12章：植被与自然环境

> 🎯 **本章目标**：掌握植被系统，学会创建逼真的自然环境

---

## 一、植被系统概述

### 1.1 什么是植被系统？

```
植被系统 = 批量放置和管理植物的工具

特点：
• 一键刷出大片森林
• 自动优化远距离显示
• 支持风动效果
• 与地形系统无缝结合
```

### 1.2 植被模式入口

```
模式工具栏 → 选择【植被】图标
或按 Shift + 4
```

---

## 二、草类型（Grass Type）

### 2.1 草类型基础

```
草类型 = 程序化生成的草地

与植被模式的区别：
• 草类型：基于地形材质，自动生成
• 植被模式：手动刷或程序化放置
```

### 2.2 创建草类型

```
1. 右键 → 植被 → Grass Type
2. 命名 GT_Grass
3. 添加静态网格体（草模型）
4. 调整密度、缩放等参数
```

---

## 三、植被工具使用

### 3.1 刷植被

```
【步骤】
1. 选择植被模式
2. 将植物资源拖到植被列表
3. 选择笔刷工具
4. 在场景中点击/拖拽绘制

【笔刷设置】
• 密度：控制数量
• 半径：控制范围
• 对齐：对齐法线/随机
```

### 3.2 程序化生成

```
【PCG基础】
UE5.2+ 新增程序化内容生成

用途：
• 按规则自动生成植被
• 基于地形高度/坡度筛选
• 避免人工刷的重复劳动
```

## 五、自然生态系统搭建

### 5.1 生态系统层次

```
完整的自然环境：

第一层：地形
├── 山脉、丘陵、平原
└── 河流、湖泊

第二层：基础植被
├── 草地（Grass Type）
├── 灌木（植被模式）
└── 树木（植被模式）

第三层：细节
├── 石头、岩石
├── 倒木、枯枝
└── 野花、蘑菇

第四层：氛围
├── 雾气
├── 粒子效果（落叶、花粉）
└── 音效（鸟鸣、风声）
```

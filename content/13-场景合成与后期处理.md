# 第13章：场景合成与后期处理

> 🎯 **本章目标**：掌握后期处理技术，营造场景氛围和视觉效果

---

## 一、后期处理概述

### 1.1 什么是后期处理？

```
后期处理 = 渲染后的"调色和特效"

类比摄影：
• 拍摄 = 场景搭建和渲染
• 后期处理 = Photoshop调色、加滤镜

在UE中：
• 调色（Color Grading）
• 泛光（Bloom）
• 雾效（Fog）
• 景深（Depth of Field）
• 抗锯齿（Anti-Aliasing）
```

### 1.2 后期处理体积

```
放置Actor → 视觉效果 → 后期处理体积

特点：
• 可以覆盖整个场景或特定区域
• 可叠加多个体积
• 支持混合过渡
```

---

## 二、常用后期效果

### 2.1 曝光与色调映射

```
【曝光设置】
• 曝光模式：自动/手动
• 曝光补偿：调整整体亮度
• 直方图：观察亮度分布

【色调映射】
• 胶片（Film）：电影感
• ACES：行业标准
• 可自定义LUT
```

### 2.2 色彩校正

```
【Color Grading】
• 全局：整体色调
• 阴影：暗部调色
• 中间调：灰部调色
• 高光：亮部调色

【LUT】
• 导入外部调色预设
• 快速实现特定风格
```

### 2.3 泛光（Bloom）

```
Bloom = 高光溢出效果

应用：
• 太阳光晕
• 霓虹灯发光
• 魔法特效

设置：
• 强度：发光程度
• 阈值：多亮才发光
• 尺寸：光晕大小
```

## 三、体积雾与大气

### 3.1 指数高度雾

```
放置Actor → 视觉效果 → 指数高度雾

参数：
• 密度：雾气浓度
• 高度衰减：随高度变化
• 颜色：雾气颜色
• 定向光散射：丁达尔效应
```

### 3.2 体积雾

```
体积雾 = 3D空间中的雾

效果：
• 光束穿透雾气
• 局部雾效
• 配合聚光灯使用
```

### 3.3 天空大气

```
Sky Atmosphere = 物理正确的大气

设置：
• 太阳高度：时间变化
• 大气密度
• 雾霾程度
```

## 四、场景氛围营造

### 4.1 时间段氛围

| 时间 | 特点 | 后期调整 |
|------|------|----------|
| 清晨 | 冷色、低角度 | 蓝调、长阴影 |
| 正午 | 明亮、高对比 | 中性色、清晰 |
| 黄昏 | 暖色、柔和 | 橙红、低对比 |
| 夜晚 | 冷色、高对比 | 蓝紫、高曝光 |

### 4.2 天气氛围

| 天气 | 特点 | 后期调整 |
|------|------|----------|
| 晴天 | 清晰、高对比 | 低雾、高饱和 |
| 阴天 | 柔和、低对比 | 高雾、低饱和 |
| 雾天 | 朦胧、低可视 | 高密度雾 |
| 雨天 | 湿润、反光 | 增加反射、蓝色调 |

## 一、虚幻引擎是什么？

### 1.1 通俗比喻

想象你是一位**舞台设计师**：
- **传统方式**：用纸笔画设计图，再实物搭建
- **用虚幻引擎**：在电脑上直接搭3D场景，实时看到最终效果，还能"走进"场景里看

**虚幻引擎（Unreal Engine，简称UE）** 就是一套超级强大的**3D场景搭建工具**，最开始是做游戏的，现在用来做：

| 应用领域 | 典型应用 | 案例 |
|---------|---------|------|
| 🎮 游戏开发 | 3A大作、独立游戏 | 《黑神话：悟空》《堡垒之夜》 |
| 🎬 影视动画 | 虚拟制片、实时渲染 | 《曼达洛人》虚拟场景 |
| 🏗️ 建筑可视化 | 建筑漫游、VR看房 | 房地产项目展示 |
| 🚗 汽车设计 | 新车展示、配置器 | 宝马、奥迪虚拟展厅 |
| 📺 虚拟直播 | 虚拟偶像、演唱会 | 虚拟主播、线上演唱会 |

## 二、为什么选择UE5做地编？

### 2.1 地编是做什么的？

**地编 = 地形/场景编辑师（Level/Environment Artist）**

就像电影需要美术布景，游戏需要有人搭建虚拟世界：

```
┌─────────────────────────────────────────────────────────┐
│                      地编工作内容                        │
├─────────────────────────────────────────────────────────┤
│                                                          │
│   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐ │
│   │   地形设计   │  │   场景搭建   │  │   氛围营造   │ │
│   ├──────────────┤  ├──────────────┤  ├──────────────┤ │
│   │ • 山脉丘陵   │  │ • 摆放建筑   │  │ • 布置光照   │ │
│   │ • 河流湖泊   │  │ • 布置道具   │  │ • 后期调色   │ │
│   │ • 道路路径   │  │ • 规划区域   │  │ • 特效添加   │ │
│   │ • 植被覆盖   │  │ • 优化布局   │  │ • 整体构图   │ │
│   └──────────────┘  └──────────────┘  └──────────────┘ │
│                                                          │
│   核心能力：审美 + 技术 + 效率                           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 2.2 UE5 地编的优势

| 特性 | 说明 | 生活比喻 |
|------|------|----------|
| **Lumen** | 实时动态光照 | 就像房间里真实开灯，光线会自动反射照亮暗处 |
| **Nanite** | 超高精度模型 | 以前用低模"欺骗"眼睛，现在直接放亿级面数模型 |
| **地形工具** | 一键生成山川河流 | 像玩"我的世界"一样 sculpt 地形 |
| **植被系统** | 批量生成森林草地 | 种一棵树，自动生成整片森林 |
| **材质系统** | 可视化节点编辑 | 像连电路一样连节点做效果 |

```
┌─────────────────────────────────────────────────────────┐
│                    游戏引擎对比                          │
├─────────────────┬─────────────────┬─────────────────────┤

├─────────────────┼─────────────────┼─────────────────────┤
│ • 画面最强      │ • 灵活轻便      │ • 轻量免费          │
│ • 地编工具强    │ • 移动端强      │ • 适合小项目        │
│ • 影视行业首选  │ • 2D游戏友好    │ • 学习曲线平缓      │
│ • 大公司首选    │ • 独立游戏多    │                     │
├─────────────────┼─────────────────┼─────────────────────┤
│ ✅ 地编推荐     │   游戏开发可选  │    入门/个人项目    │
└─────────────────┴─────────────────┴─────────────────────┘
```

**为什么地编选UE5？**
1. **行业需求大**：游戏公司、影视公司、建筑可视化都在招UE地编
2. **效果好**：出图质量高，作品容易"唬人"
3. **生态成熟**：教程多、资源多、招人标准统一
4. **薪资可观**：初级地编8-15K，高级地编20-40K+

## 四、地编岗位能力要求

### 4.1 初级地编（0-2年）

```
必备技能：
├── UE基础操作（视图、导航、摆放）
├── 地形系统（sculpting、材质层、绘制）
├── 基础材质理解（metallic/roughness 工作流）
├── 灯光布置（三点布光、氛围营造）
├── 资源管理（导入、整理、优化意识）
└── 基础后期处理（调色、雾效、体积光）
```

### 4.2 中级地编（2-4年）

```
进阶技能：
├── 复杂地形 sculpting（山脉、峡谷、岛屿）
├── 高级材质（地形混合、顶点绘制、视差）
├── Lumen + Nanite 高级应用
├── World Partition 大世界管理
├── PCG（程序化生成）基础
├── 性能优化与调试
└── 团队协作与版本控制
```

### 4.3 常用软件配合

| 软件 | 用途 | 必须程度 |
|------|------|----------|
| **UE5** | 主战场 | ⭐⭐⭐⭐⭐ |

| **Photoshop** | 贴图处理 | ⭐⭐⭐⭐ |

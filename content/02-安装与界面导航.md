# 第2章：安装与界面导航

> 🎯 **本章目标**：成功安装UE5，熟悉主界面各个区域的功能与操作

---

## 一、安装虚幻引擎

### 1.1 系统要求

#### 最低配置 vs 推荐配置

| 配置项 | 最低配置 | 推荐配置 | 说明 |
|--------|----------|----------|------|
| **操作系统** | Windows 10 64位 | Windows 11 64位 | 必须64位 |
| **处理器** | 四核处理器 | 十二核以上 | 编译光照需要多核 |
| **内存** | 8 GB | 32 GB+ | 大场景非常吃内存 |
| **显卡** | GTX 1060 6GB | RTX 3060+ | 显存越大越好 |
| **硬盘** | 15GB 空间 | SSD 100GB+ | **SSD是必须的！** |
| **显示器** | 1080p | 2K/4K | 大屏效率更高 |

💡 **地编建议**：
- **SSD是必须的！** 加载大场景机械硬盘会卡死
- **内存越大越好**，32GB是甜点，64GB更舒服
- **显存要充足**，8GB起步，12GB+更好

### 1.2 下载 Epic Games Launcher

```
步骤：
1. 访问官网：https://store.epicgames.com/zh-CN/
2. 点击右上角"获取 Epic Games 启动器"
3. 下载并安装（安装在默认位置即可）
```

### 1.3 安装 UE5 详细步骤

```
打开 Epic Games Launcher
       │
       ▼
┌────────────────────────────────────────┐
│  左侧菜单 → 选择【虚幻引擎】标签页      │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  点击【库】→ 引擎版本旁的【+】按钮      │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  选择 【5.7 版本】（或当前最新稳定版）  │
│  勾选 【默认组件】即可                  │
│  （初学者不需要额外勾选）               │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  选择安装路径 → 点击安装               │
│  💡 建议装在SSD上                      │
└────────────────────────────────────────┘
```

### 1.4 安装组件说明

| 组件 | 是否勾选 | 说明 |
|------|----------|------|
| **核心组件** | ✅ 必须 | 引擎本体 |
| **初学者内容包** | ✅ 建议 | 示例资源，学习用 |
| **模板和功能包** | ✅ 建议 | 项目模板 |
| **引擎源码** | ❌ 可选 | 不需要，C++开发才用 |
| **目标平台** | 按需 | 只做PC可不勾 |
| **IDE支持** | ❌ 可选 | 有编程需求再装 |

### 3.2 五大核心区域详解

#### ① 主视口（Viewport）- 你的舞台

这是你最常待的地方，就像摄影师的取景器：

```
┌──────────────────────────────────────────┐
│  ┌┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┐  │
│  ├┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┤  │ ← 透视图/顶视图/侧视图切换
│  │                                  │  │
│  │         3D 场景视图               │  │
│  │                                  │  │
│  │    (这里显示你搭建的场景)          │  │
│  │                                  │  │
│  │                                  │  │
│  │  x - 选中物体的变换控件            │  │
│  │     (移动/旋转/缩放)              │  │
│  │                                  │  │
│  │                                  │  │
│  └──────────────────────────────────┘  │
│     [透视] [光照] [显示] [游戏视图]      │ ← 视图选项
└──────────────────────────────────────────┘
```

**视图模式切换**：
- **Alt + G** = 游戏视图（隐藏编辑器图标）
- **Alt + 2** = 线框模式
- **Alt + 3** = 无光照模式
- **Alt + 4** = 光照模式（常用）
- **Alt + 5** = 细节光照

#### ② 大纲视图（Outliner）- 场景目录

```
┌─────────────────┐
│  🔍 搜索...      │
├─────────────────┤
│  PersistentLevel│ ← 当前关卡
│  ├─ Light Source│ ← 光源
│  ├─ SkyLight    │ ← 天光
│  ├─ PlayerStart │ ← 玩家出生点
│  ├─ Floor       │ ← 地板
│  └─ ...         │
│                 │
│ [+] 添加新对象   │
└─────────────────┘
```

**常用操作**：
- **单击** = 选中物体
- **双击** = 选中并聚焦（F键效果）
- **右键** = 菜单操作
- **拖拽** = 调整层级（创建父子关系）

#### ③ 细节面板（Details）- 属性编辑器

```
┌─────────────────────────┐
│  选中对象: Floor        │
├─────────────────────────┤
│ ▼ Transform 变换        │
│   Location 位置   x y z │
│   Rotation 旋转   x y z │
│   Scale    缩放   x y z │
├─────────────────────────┤
│ ▼ Static Mesh 静态网格   │
│   [选择网格体...]        │
│   [圆柱体]              │
├─────────────────────────┤
│ ▼ Materials 材质        │
│   [0] M_Basic_Floor     │
├─────────────────────────┤
│ ▼ Lighting 光照         │
│   Cast Shadow ☑         │
│   ...                   │
└─────────────────────────┘
```

💡 **重要**：面板右上角可以**锁定**，防止切换选择时变化

#### ④ 内容浏览器（Content Browser）- 资源仓库

```
┌────────────────────────────────────┐
│ 📁 /Game/  面包屑导航  [搜索框]     │
├────────────────────────────────────┤
│ ┌─────────────────────────────────┐│
│ │                                 ││
│ │  【文件夹树】    【资源网格视图】 ││
│ │                                 ││
│ │  📁 Content     ┌────┐┌────┐   ││
│ │  ├─ 📁 Maps    │📦  ││🔵  │   ││
│ │  ├─ 📁 Meshes  │SM  ││M   │   ││
│ │  └─ 📁 ...     └────┘└────┘   ││
│ │                      ...       ││
│ │                                 ││
│ └─────────────────────────────────┘│
├────────────────────────────────────┤
│ [导入] [新增] [保存全部]            │
└────────────────────────────────────┘
```

**资源类型图标**：
- 📁 = 文件夹
- 📦 = 静态网格体（模型）
- 🔵 = 材质/材质实例
- 🔴 = 贴图
- 💡 = 蓝图
- 🗺️ = 关卡

#### ⑤ 工具栏 - 快捷操作

```
┌────────────────────────────────────────────────────────────┐
│ [💾] [↩] [↪] │ [选择] [地形] [植被] [笔刷] │ [▶播放] │ ⚙️  │
└────────────────────────────────────────────────────────────┘
         │        │      │      │      │
         │        │      │      │      └─ 模式切换
         │        │      │      └─ 植被绘制模式
         │        │      └─ 地形编辑模式
         │        └─ 默认选择模式
```

## 四、窗口布局自定义

### 4.1 保存自定义布局

```
窗口 → 加载布局 → 保存布局 → 命名"地编工作区"
```

### 4.2 推荐地编布局

```
┌──────────────────────────────────────────────────────────────┐
│  菜单栏 + 主工具栏                                            │
├──────────────┬──────────────────────────────────┬────────────┤
│              │                                  │            │
│  大纲视图    │          主视口（大）            │  细节面板  │

│              │                                  │            │
```

**布局调整技巧**：
- 拖拽面板标题栏可以移动位置

- 双击面板标题栏可以最大化
- 右键面板标题栏有更多选项

## 五、项目设置优化

### 5.1 编辑器偏好设置

```
编辑 → 编辑器偏好设置

【通用】
  自动保存：开启，间隔300秒
  自动保存目录：项目/Saved/Autosaves

【视口】
  使用飞行速度缩放：开启
  默认相机速度：4（可根据喜好调整）

【加载和保存】
  启动时加载上一个项目：建议开启
  自动重新导入：开启（外部修改文件自动更新）
```

### 5.2 项目设置

```
编辑 → 项目设置

【地图和模式】
  默认地图 → 编辑器开始地图
  默认地图 → 游戏默认地图

【渲染】
  动态全局光照：Lumen（UE5默认）
  反射：Lumen反射
  阴影：虚拟阴影贴图

【平台】Windows
  默认RHI：DirectX 12
```

# 🎮 虚幻引擎5 系统学习手册

> **目标人群**：完全零基础、无引擎经验、想系统学习UE5的新手  
> **核心目的**：从零基础到工程实践，搭建完整学习路径  
> **风格定位**：通俗易懂、图文并茂、理论与实践结合

**依赖关系说明**：
- **基础层**：项目/关卡 → 视口/内容浏览器/静态网格体（必须先掌握）
- **核心层**：材质系统是所有视觉效果的基础
- **应用层**：地形/光照/植被 可以并行学习，但都依赖材质
- **整合层**：后期处理将所有效果合成最终画面
- **支撑层**：Lumen和Nanite贯穿全程，提供技术支撑

## 📊 学习路径建议

### 纯小白路线（零基础）
```
第1-2周：入门篇（第1-5章）
   └── 每天1-2小时，熟悉界面和基本操作

第3-6周：进阶篇（第6-11章）
   └── 每天2-3小时，重点攻克材质和光照

第7-10周：高级篇（第12-14章）+ 实战项目
   └── 每天3小时+，动手做完整场景
```

### 有3D基础路线
```
第1周：快速浏览入门篇
   └── 重点看界面差异和UE特性

第2-4周：进阶篇（第6-11章）
   └── 重点学习Lumen、Nanite等UE5新特性

第5-8周：高级篇 + 实战
   └── 做1-2个完整场景作品
```

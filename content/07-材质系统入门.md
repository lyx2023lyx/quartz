# 第7章：材质系统入门

> 🎯 **本章目标**：理解材质原理，学会创建和编辑基础材质，掌握PBR工作流程

---

## 一、什么是材质？

### 1.1 通俗解释

**材质 = 物体的"皮肤"**

```
想象现实世界：

同一把椅子，不同材质：
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   木椅子    │  │   铁椅子    │  │   塑料椅子  │
├─────────────┤  ├─────────────┤  ├─────────────┤
│ • 棕色      │  │ • 银灰色    │  │ • 五颜六色  │
│ • 有木纹    │  │ • 反光      │  │ • 哑光      │
│ • 不反光    │  │ • 冰冷感    │  │ • 光滑      │
│ • 粗糙      │  │ • 光滑      │  │ • 轻飘感    │
└─────────────┘  └─────────────┘  └─────────────┘

在UE中，材质决定：
├── 颜色（Color）
├── 反光度（Metallic/Roughness）
├── 纹理（Texture）
└── 特殊效果（发光、透明等）
```

### 1.2 材质与网格体的关系

```
网格体 = 形状（Geometry）
   ↓
材质 = 外观（Appearance）
   ↓
最终渲染效果

类比：
• 网格体 = 石膏像的模具
• 材质 = 给石膏像上色、喷漆
• 两者结合 = 完整的雕像
```

## 二、PBR 基于物理的渲染

### 2.1 什么是PBR？

```
PBR = Physically Based Rendering = 基于物理的渲染

简单说：材质参数符合真实物理规律

【Metallic/Roughness 工作流】
                    │
        ┌───────────┼───────────┐
        │           │           │
        ▼           ▼           ▼
   ┌─────────┐ ┌─────────┐ ┌─────────┐
   │BaseColor│ │Metallic │ │Roughness│
   │基础颜色 │ │金属度   │ │粗糙度   │
   └─────────┘ └─────────┘ └─────────┘
        │           │           │
        │    ┌──────┴──────┐    │
        │    │             │    │
        └────┤    材质      ├────┘
             │   (输出)    │
             └─────────────┘

三个核心参数：
• BaseColor = 物体本身的颜色（木头是棕色，铁是灰色）
• Metallic = 是否是金属（0=塑料/木头，1=金/银/铁）
• Roughness = 表面粗糙度（0=镜子，1=磨砂）
```

### 2.2 PBR 核心参数详解

#### Base Color（基础颜色）

| 材质类型 | BaseColor特征 | 示例值 |
|----------|---------------|--------|
| 木头 | 深棕色，无高光 | (0.2, 0.1, 0.05) |
| 石头 | 灰色/土色 | (0.3, 0.3, 0.28) |
| 草地 | 绿色 | (0.1, 0.4, 0.1) |
| 金属 | 灰度值（反射颜色） | (0.8, 0.8, 0.8) |

#### Metallic（金属度）

```
Metallic = 0（非金属）    Metallic = 1（金属）
    ↓                        ↓
┌──────────┐            ┌──────────┐
│ 塑料     │            │ 黄金     │
│ 木头     │            │ 钢铁     │
│ 石头     │            │ 铜       │
│ 混凝土   │            │ 铝       │
└──────────┘            └──────────┘

💡 绝大多数材质非0即1，很少用中间值
```

#### Roughness（粗糙度）

```
Roughness 0.0          Roughness 0.5          Roughness 1.0
    ↓                      ↓                      ↓
┌──────────┐          ┌──────────┐          ┌──────────┐
│  镜子    │          │  混凝土  │          │  天鹅绒  │
│  光滑金属│          │  粗糙石头│          │  泥土    │
│  玻璃    │          │  皮肤    │          │  粉笔    │
└──────────┘          └──────────┘          └──────────┘

0 = 完全光滑（镜面反射）
1 = 完全粗糙（无光泽）
```

### 3.1 打开材质编辑器

```
方式1：创建新材质
内容浏览器 → 右键 → 材质和纹理 → 材质
命名：M_材质名（如 M_Wood_Oak）

方式2：编辑现有材质
双击材质资源

方式3：在模型上编辑
选中模型 → 细节面板 → 材质槽 → 点击材质 → 编辑

### 3.3 核心概念：节点

```
【节点 = 材质的功能模块】

每个节点做一件事，用线连起来组成材质逻辑：

输入节点                          输出节点
┌─────────────┐                   ┌─────────────┐
│  贴图样本   │                   │   结果      │
│ Texture     │───┐           ┌──→│  Result    │
│ Sample      │   │           │   └─────────────┘
└─────────────┘   │   ┌───┐   │        ↑
                  └──→│ × │───┘   最终输出到渲染
┌─────────────┐   ┌──→│   │              
│  颜色常量   │   │   └───┘               
│ Constant3   │───┘   数学节点           
└─────────────┘                       

常用节点类型：
• 常量节点 = 固定数值（如颜色0.5,0.5,0.5）
• 参数节点 = 可调数值（材质实例用）
• 贴图节点 = 导入的图片
• 数学节点 = 加减乘除（如Multiply混合）
• 函数节点 = 复杂功能（如旋转UV）
```

## 四、创建基础材质

### 4.1 纯色材质

```
【最简单的材质】纯色

步骤：
1. 创建材质 M_Solid_Red
2. 双击打开材质编辑器

6. 点击【应用】→【保存】

【带纹理的材质】木纹

3. 在图表中右键 → Texture Sample

## 五、材质属性详解

### 5.1 材质混合模式

```
┌───────────────────────────────────────────────────────┐
│                   混合模式对比                         │
├─────────────┬──────────────────┬──────────────────────┤
│ 模式        │ 用途             │ 示例                 │
├─────────────┼──────────────────┼──────────────────────┤
│ Opaque      │ 完全不透明       │ 石头、墙面、木头     │
│             │ （性能最好）     │                      │
├─────────────┼──────────────────┼──────────────────────┤
│ Masked      │ 有孔洞的不透明   │ 树叶、栅栏、铁网     │
│             │ （用Opacity Mask）│                      │
├─────────────┼──────────────────┼──────────────────────┤
│ Translucent │ 半透明           │ 玻璃、水、烟雾       │
│             │ （性能开销大）   │                      │
├─────────────┼──────────────────┼──────────────────────┤
│ Additive    │ 叠加发光         │ 火焰、能量、UI       │
│             │ （越叠越亮）     │                      │
└─────────────┴──────────────────┴──────────────────────┘

【实际应用场景】
• Opaque: 建筑墙面、地面、家具主体
• Masked: 树叶（镂空）、铁丝网、栅栏
• Translucent: 玻璃窗、水面、雾气
• Additive: 魔法特效、发光文字、能量护盾
```

### 5.2 着色模型

| 着色模型 | 特点 | 应用场景 |
|----------|------|----------|
| **Default Lit** | 标准光照模型 | 绝大多数材质 |
| **Unlit** | 不受光照影响 | 天空盒、UI、自发光 |
| **Subsurface** | 次表面散射 | 皮肤、蜡、玉石 |
| **Clear Coat** | 清漆层 | 汽车漆面、光滑塑料 |
| **Hair** | 头发专用 | 角色头发 |

### 5.3 细节面板关键设置

```
┌─────────────────────────────────────────────┐
│ Physical Material 物理材质                   │
├─────────────────────────────────────────────┤
│ 【Material】                                 │
│   Material Domain: Surface                  │
│   Blend Mode: Opaque                        │
│   Shading Model: Default Lit                │
│   Two Sided: ☐                              │
│                                             │
│ 【Translucency】（Translucent模式时显示）    │
│   Lighting Mode: Surface Translucency       │
│   Refraction: 折射率（玻璃1.5，水1.33）     │

## 六、常用材质节点

### 6.1 基础节点

| 节点名 | 用途 | 示例 |
|--------|------|------|
| **Texture Sample** | 读取贴图 | 所有贴图输入 |
| **Constant** | 单值常量 | Roughness数值 |
| **Constant3Vector** | RGB颜色 | BaseColor颜色 |
| **Constant4Vector** | RGBA | 带透明颜色 |
| **Multiply** | 乘法混合 | 贴图×颜色=染色 |
| **Add** | 加法叠加 | 颜色+发光 |
| **Lerp** | 线性插值 | 两种材质过渡 |

### 6.2 UV相关节点

```
【UV基础】
UV = 贴图坐标，告诉材质怎么把2D贴图贴到3D模型上

┌───────────────────────────┐
│     UV坐标系（0-1空间）    │
│                           │
│   (0,1)        (1,1)      │
│     ┌──────────┐          │
│     │          │          │
│     │  贴图    │          │
│     │          │          │
│     └──────────┘          │
│   (0,0)        (1,0)      │
│                           │
└───────────────────────────┘

常用UV节点：
• TextureCoordinate = 基础UV
• Panner = UV平移（流水、飘云）
• Rotator = UV旋转
• FlipBook = 序列帧动画
```

### 6.3 世界对齐节点

```
【World Aligned Texture】世界对齐贴图

问题：地形斜坡贴图会拉伸
解决：让贴图沿着世界坐标对齐，不是模型UV

用法：
World Aligned Texture → Base Color
                       ↑
                  选择贴图

效果：
┌────────────────────────────┐
│  地形：                     │
│        ╱╲                  │
│       ╱  ╲                 │
│      ╱    ╲                │
│                             │
│  普通UV：贴图跟着斜坡拉伸   │
│  世界对齐：贴图保持水平     │
└────────────────────────────┘

地编必备：地形材质用它避免拉伸！
```

## 七、材质实例

### 7.1 什么是材质实例？

```
【概念】
母材质（Master Material）= 基础配方
材质实例（Material Instance）= 基于配方的变种

类比：
母材质 = 基础奶茶配方
├── MI_奶茶_珍珠（+珍珠）
├── MI_奶茶_椰果（+椰果）
└── MI_奶茶_布丁（+布丁）

好处：
• 改母材质，所有实例都变
• 实例参数可调，不用重新编译
• 性能优化（实例渲染更快）
```

### 7.2 创建材质实例

```
步骤：
1. 创建母材质 M_Master_Wall
2. 设置需要暴露的参数（右键 → 转换为参数）
3. 保存母材质
4. 右键母材质 → 创建材质实例
5. 命名 MI_Wall_Red、MI_Wall_Blue

参数类型：
┌────────────────────────────────┐
│ Scalar Parameter 标量参数      │
│   • 数字（0-1）                │
│   • 如：Roughness值            │
├────────────────────────────────┤
│ Vector Parameter 向量参数      │
│   • 颜色（RGB）                │
│   • 如：BaseColor颜色          │
├────────────────────────────────┤
│ Texture Parameter 贴图参数     │
│   • 可更换的贴图               │
│   • 如：Diffuse贴图            │
└────────────────────────────────┘
```

### 7.3 材质实例编辑器

```
材质实例界面（简化版）：

┌─────────────────────────────────────────────┐
│ Parent: M_Master_Wall                       │

│   ☑ Override Base Color                    │
│   Base Color: [红色选择器] ⬜              │
│                                             │
│   ☑ Override Roughness                     │

│                                             │
│   ☑ Override Diffuse Texture               │

## 八、地编常用材质类型

### 8.1 地形混合材质

```
【地形需要混合多种材质】

地形材质特点：
• 草地、泥土、石头混合
• 用Weight Map控制混合
• 每层可以独立调整

节点结构：
┌───────────────────────────────────────┐
│ Layer 1: 草地                          │
│ Layer 2: 泥土                          │
│ Layer 3: 石头                          │
│         ↓                             │
│    [Landscape Layer Blend]             │
│         ↓                             │
│     混合输出 → Base Color              │
└───────────────────────────────────────┘

每层包含：
• Albedo（颜色）
• Normal（法线）
• Roughness（粗糙度）
```

```
【Vertex Paint = 顶点绘制】

用途：在模型上手动涂抹不同材质

例子：

│  墙面：                        │

│  │ 砖 ████ 石头    │          │

│  │ 砖 ████ 石头    │          │

│                                │
│  在编辑器里用笔刷涂抹：         │

│  • 蓝色通道 = 苔藓             │

材质节点：Vertex Color → Lerp（混合多个层）
```

## 九、材质优化

### 9.1 性能原则

```
┌───────────────────────────────────────────────────────┐
│                  材质性能优化                          │
├───────────────────────────────────────────────────────┤
│                                                        │
│ 【贴图尺寸】                                           │
│ • 尽量用2的幂次方（512、1024、2048）                  │
│ • 远处物体用小贴图（512）                              │
│ • 近处重要物体用大贴图（2048）                         │
│ • 避免用奇数尺寸（如1000×1000）                       │
│                                                        │
│ 【贴图压缩】                                           │
│ • 颜色贴图：DXT1/BC1（压缩高）                         │
│ • 法线贴图：DXT5/BC5（保留细节）                       │
│ • UI贴图：不压缩（保持锐利）                           │
│                                                        │
│ 【材质复杂度】                                         │
│ • 节点越少越好                                         │
│ • 少用复杂数学计算                                     │
│ • 半透明材质尽量少用（性能杀手）                       │
│                                                        │
│ 【Draw Call】                                          │
│ • 相同材质合并（Static Mesh合并）                      │
│ • 使用材质实例而不是多个母材质                         │
│                                                        │
└───────────────────────────────────────────────────────┘
```

### 9.2 检查清单

```
✅ 材质发布前检查：

□ 贴图已经压缩？
□ 不需要的贴图已经移除？
□ Blend Mode选择正确？
□ Two Sided只在需要时开启？
□ 半透明材质数量合理？
□ 材质实例参数命名清晰？
□ 预览中效果正确？
```

## 一、光照基础概念

### 1.1 为什么光照很重要？

```
同一场景，不同光照：

┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│    晴朗正午     │  │    黄昏日落     │  │    阴森夜晚     │
├─────────────────┤  ├─────────────────┤  ├─────────────────┤
│ ☀️ 明亮刺眼     │  │ 🌅 暖色柔和     │  │ 🌙 冷色神秘     │
│ 阴影清晰        │  │ 长阴影          │  │ 强烈对比        │
│ 高饱和度        │  │ 金黄色调        │  │ 蓝紫色调        │
│ 活力/明亮       │  │ 温馨/浪漫       │  │ 恐怖/神秘       │
└─────────────────┘  └─────────────────┘  └─────────────────┘

光照决定场景的情绪和氛围！
```

### 1.2 光源类型速览

| 光源类型 | 特点 | 地编用途 | 图标 |
|----------|------|----------|------|
| **Directional Light** | 平行光，模拟太阳光 | 主光源、户外场景 | ☀️ |
| **Point Light** | 点光源，向四周发光 | 灯泡、火把、发光物 | 💡 |
| **Spot Light** | 聚光灯，锥形范围 | 手电筒、舞台灯、路灯 | 🔦 |
| **Rect Light** | 矩形面光源，柔和 | 窗户光、灯箱、柔光 | ⬜ |
| **Sky Light** | 天空光，环境填充 | 填充阴影、天空调色 | 🌤️ |

## 二、Directional Light 方向光

### 2.1 什么是方向光？

```
【通俗理解】
方向光 = 太阳光（或月光）

特点：
• 光线平行（因为太阳太远）
• 没有具体位置，只有方向
• 照亮整个场景
• 产生平行阴影

┌─────────────────────────────────────────────┐
│                                             │
│    \\  \\  \\  \\  \\  \\  \\  \\  \\         │
│     \\  \\  \\  \\  \\  \\  \\  \\  \\        │
│      \\  \\  \\  \\  \\  \\  \\  \\  \\       │
│       \\  \\  \\  \\  \\  \\  \\  \\  \\      │
│        ═══════════════════════             │
│              地面                           │
│                                             │
│  所有光线平行，方向一致                      │
│  就像太阳光照到地球                          │
└─────────────────────────────────────────────┘
```

### 2.2 方向光属性

```
选中 DirectionalLight → 细节面板

┌─────────────────────────────────────────────┐
│ Light                                         │
├─────────────────────────────────────────────┤
│ Intensity 强度                               │
│   •  lux（勒克斯）物理单位                   │
│   • 室外晴天：10000-100000 lux              │
│   • 室内灯光：100-1000 lux                  │
│   • 月光：0.1-1 lux                         │
├─────────────────────────────────────────────┤
│ Light Color 光色                             │
│   • 正午：白色/淡黄（6500K）                │
│   • 日落：橙红（3000K）                     │
│   • 阴天：蓝灰（8000K）                     │
├─────────────────────────────────────────────┤
│ Temperature 色温 ☑                           │
│   • 勾选后按色温调整                         │
│   • 数值越低越暖（红），越高越冷（蓝）      │
├─────────────────────────────────────────────┤
│ Cast Shadows 投射阴影 ☑                      │
│   • 必须勾选！否则没有阴影                  │
└─────────────────────────────────────────────┘
```

### 2.3 操作技巧

```
【改变太阳方向】
• 选中方向光
• 旋转它（E键旋转模式）
• 观察阴影方向变化

【太阳位置可视化】
• 方向光绑定 SkyAtmosphere
• 旋转方向光，天空中的太阳会跟着动
• 同时影响时间氛围（正午/黄昏/夜晚）

【快捷键】
• Ctrl + L + 鼠标移动 = 快速改变光照方向
  （类似Maya的灯光操作）
```

## 三、Point Light 点光源

### 3.1 点光源原理

```
【通俗理解】
点光源 = 灯泡、火把、蜡烛

特点：
• 从一个点向四周发光
• 光线随距离衰减
• 强度随距离平方衰减（真实物理）

┌─────────────────────────────────────────────┐
│                                             │
│              ● 点光源                       │
│             /|\                             │
│            / | \                            │
│           /  |  \                           │
│          /   |   \                          │
│         /    |    \                         │
│        /     |     \                        │
│       ═══════╧═══════                      │
│              地面                           │
│                                             │
│  光线360度散射                              │
│  越近越亮，越远越暗                         │
└─────────────────────────────────────────────┘
```

### 3.2 点光源属性

```
┌─────────────────────────────────────────────┐
│ Point Light 属性                             │
├─────────────────────────────────────────────┤
│ Intensity 强度                               │
│   • 单位：流明（lm）或坎德拉（cd）          │
│   • 灯泡：100-1000 lm                       │
│   • 火把：500-2000 lm                       │
│   • 路灯：1000-5000 lm                      │
├─────────────────────────────────────────────┤
│ Attenuation Radius 衰减半径                  │
│   • 光能照到的最远距离                       │
│   • 黄色线球显示范围                         │
│   • 设置合适大小（不要太大浪费性能）        │
├─────────────────────────────────────────────┤
│ Cast Shadows 投射阴影 ☑                      │
│   • 动态物体投阴影                           │
│   • 静态物体建议烘焙阴影                     │
└─────────────────────────────────────────────┘
```

### 3.3 使用技巧

```
【室内布光示例】

房间布光：
┌───────────────────────────────────────┐
│  💡                                    │
│  天花板灯（点光源）                    │
│        │                               │
│        ▼                               │
│  ┌─────────────┐                       │
│  │             │  🕯️                   │
│  │    房间     │  桌子上的蜡烛         │
│  │             │  （暖色小半径）       │
│  │      🚪    │                       │
│  └─────────────┘                       │
│                                        │
│  窗户透进的Directional Light           │
└───────────────────────────────────────┘

【性能优化】
• 控制Attenuation Radius不要过大
• 重叠的光源不要超过3-4个
• 静态物体用烘焙代替实时光
```

## 四、Spot Light 聚光灯

### 4.1 聚光灯原理

```
【通俗理解】
聚光灯 = 手电筒、舞台灯、车灯、探照灯

特点：
• 有明确方向
• 锥形照射范围
• 内锥：最亮区域
• 外锥：衰减过渡区

┌─────────────────────────────────────────────┐
│                                             │
│              ● 聚光灯                       │
│             /\                              │
│            /  \  ← 外锥（半影）             │
│           /    \                            │
│          /  __  \                           │
│         /  /  \  \  ← 内锥（本影）          │
│        /  /    \  \                         │
│       ══╧══════╧═══                        │
│              地面                           │
│                                             │
│  光线集中在锥形范围内                        │
│  内圈亮，外圈渐暗                           │
└─────────────────────────────────────────────┘
```

### 4.2 聚光灯属性

```
┌─────────────────────────────────────────────┐
│ Spot Light 特有属性                          │
├─────────────────────────────────────────────┤
│ Inner Cone Angle 内锥角度                    │
│   • 光线最亮的中心区域                       │
│   • 范围：0-90度                             │
│   • 手电筒：10-20度                         │
│   • 舞台灯：30-45度                         │
├─────────────────────────────────────────────┤
│ Outer Cone Angle 外锥角度                    │
│   • 光线衰减到零的边界                       │
│   • 必须 > 内锥角度                         │
│   • 决定光锥的柔和度                         │
└─────────────────────────────────────────────┘
```

### 4.3 应用场景

| 场景 | Inner | Outer | Intensity | 备注 |
|------|-------|-------|-----------|------|
| 手电筒 | 15° | 30° | 5000 | 锐利光束 |
| 路灯 | 30° | 60° | 3000 | 投射阴影 |
| 舞台灯 | 20° | 40° | 10000 | 暖色 |
| 探照灯 | 5° | 15° | 50000 | 锐利光束 |

## 五、Sky Light 天空光

### 5.1 为什么需要天空光？

```
【问题】
只用DirectionalLight的问题：
• 阴影处全黑（不真实）
• 没有天空的颜色反射

【天空光的作用】
模拟天空/环境对场景的间接光照：
• 阴影处不是全黑
• 获得天空的颜色
• 整体更真实自然

┌─────────────────────────────────────────────┐
│  光源 💡                                    │
│    │                                        │
│    │ 直接光                                 │
│    ▼                                        │
│  ┌────┐  → 直接照亮                        │
│  │红墙│                                    │
│  └─┬──┘                                    │
│    │                                        │
│    │ 反射光（Sky Light捕获）               │
│    ▼                                        │
│  ┌────┐  → 阴影处也被照亮                  │
│  │白球│  （不再全黑）                      │
│  └────┘                                    │
│                                             │
│  Sky Light 让阴影有细节，场景更真实        │
└─────────────────────────────────────────────┘
```

### 5.2 天空光设置

```
┌─────────────────────────────────────────────┐
│ Sky Light 属性                               │
├─────────────────────────────────────────────┤
│ Intensity Scale 强度缩放                     │
│   • 默认1.0                                  │
│   • 调整整体环境光亮度                       │
├─────────────────────────────────────────────┤
│ Real Time Capture 实时捕获 ☑                 │
│   • 动态更新天空光                           │
│   • 配合SkyAtmosphere自动更新                │
│   • UE5推荐开启                              │
└─────────────────────────────────────────────┘
```

## 六、光照移动性（Mobility）

### 6.1 三种移动性对比

| 移动性 | 特点 | 适用场景 | 性能 |
|--------|------|----------|------|
| **Static** | 静态，烘焙光照 | 不动的一切 | 最好 |
| **Stationary** | 半静态，可变色 | 会开关的灯 | 好 |
| **Movable** | 完全动态 | 移动的光源 | 开销大 |

### 6.2 如何选择

```
【决策流程】

这个光源会移动位置吗？
        │
        ├── 是 → Movable 可移动
        │         （手电筒、车灯）
        │
        └── 否 → 位置固定
                  │
                  ├── 需要改颜色/亮度吗？
                  │       │
                  │       ├── 是 → Stationary 固定
                  │       │         （可开关的灯）
                  │       │
                  │       └── 否 → Static 静态
                  │                 （太阳光）

【地编建议】
• 太阳光：Static（烘焙）或 Stationary
• 室内灯：Stationary
• 玩家手电筒：Movable
• 特效光：Movable
```

## 七、光照烘焙（Build Lighting）

### 7.1 什么是烘焙？

```
【通俗理解】
烘焙 = 预计算光照

实时计算：
• 每帧都算光影
• 性能开销大
• 适合动态物体

烘焙：
• 提前算好光影，存成贴图
• 运行时直接读取
• 性能极好，质量更高
• 只有静态物体能烘焙

【对比】
实时计算：                         烘焙后：
┌─────────────┐                  ┌─────────────┐
│  ┌─────┐    │                  │  ┌─────┐    │
│  │ 墙  │    │ 每帧计算         │  │▓▓▓▓▓│    │ 读取贴图
│  └─────┘    │ 性能消耗大       │  └─────┘    │ 性能极好
│             │                  │             │
└─────────────┘                  └─────────────┘
```

### 7.2 烘焙操作

```
【烘焙光照】

菜单栏 → 构建 → 仅构建光照
或快捷键：Ctrl + Shift + 分号

构建选项：
┌─────────────────────────────────────────────┐
│ 构建质量：                                   │
│ • 预览（Preview）- 快速，质量低             │
│ • 中等（Medium）- 平衡                      │
│ • 高（High）- 质量好，耗时                  │
│ • 产品级（Production）- 最高质量            │
├─────────────────────────────────────────────┤
│ 构建什么：                                   │
│ ☑ 光照（Lighting）                          │
│ ☐ 几何体（Geometry）                        │
│ ☐ 导航（Navigation）                        │
│ ☐ 全部（Build All）                         │
└─────────────────────────────────────────────┘

💡 开发时用【预览】，发布前用【产品级】
```

## 八、场景布光实战

### 8.1 三点布光法

```
【经典布光方案】

        轮廓光/背光
              ↓
            💡
           / \
          /   \
         /  📷 \
    💡──┼───────┼──💡
   主光      物体      补光

【参数参考】
• 主光（Key）：最亮，决定主阴影方向
• 补光（Fill）：亮度50-70%，填充阴影
• 轮廓光（Rim）：亮度100%+，勾勒边缘
```
